# PhotoRestore.ga — Agent Configuration

## Identité du projet
SaaS de restauration de photos anciennes pour le marché gabonais
et francophone. Stack : Next.js 14/15, Supabase, Tailwind CSS,
Google Generative AI. Marché cible : Gabon, Afrique francophone.

## Règles absolues
- NEVER toucher au code d'éditeur ou de génération PDF sans autorisation
- ALWAYS faire un plan écrit avant toute implémentation
- ALWAYS vérifier les skills disponibles dans .claude/skills/ avant
  toute tâche complexe
- NEVER chaîner plusieurs fichiers dans une seule tâche
- Une tâche = un fichier = une confirmation avant de continuer
- Toujours répondre en français

## Avant chaque tâche complexe
1. Lire les skills disponibles dans .claude/skills/
2. Identifier lequel s'applique
3. Présenter un plan d'implémentation en 3-5 étapes
4. Attendre la validation avant de coder

## Stack technique
- Framework : Next.js 14/15 App Router
- Auth + DB : Supabase
- Styles : Tailwind CSS + shadcn/ui
- IA : Google Generative AI (Gemini)
- Paiement : Orange Money, Wave, Carte bancaire
- Déploiement : Vercel

## Design System
- Style : Premium, professionnel, moderne
- Inspiration : Grandes SaaS mondiaux (Linear, Vercel, Stripe)
- Public : Marché africain francophone
- Palette : À définir avec AIDesigner MCP

## MCPs disponibles
- Supabase MCP : pour toutes les requêtes base de données
- AIDesigner MCP : pour la génération UI/UX premium (à installer)
